# AI-Powered Multi-Input Article Summarizer

An AI-powered Natural Language Processing (NLP) application that generates concise summaries from long articles using the Facebook BART Transformer model.

## Features

- Text Summarization
- PDF Summarization
- Website Article Summarization
- Transformer-based Abstractive Summarization
- Automatic Text Extraction
- Easy-to-use Python Implementation

## Technologies Used

- Python
- Hugging Face Transformers
- PyTorch
- BART (facebook/bart-large-cnn)
- Newspaper3k
- PyPDF2

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Powered-Article-Summarizer.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```

## Workflow

1. Input Text / PDF / URL
2. Extract Text
3. Process using BART Transformer
4. Generate Summary
5. Display Result
